## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* pandas
  ```sh
  pip install pandas
  ```
* numpy
  ```sh
  pip install numpy
  ```
* matplotlib
  ```sh
  pip install matplotlib
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/reoyamanaka/cost_visualizer.git
   ```
2. Install requirements
     ```sh
     pip install -r requirements.txt
     ```
